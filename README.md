# My Express App

## Deskripsi

Proyek ini adalah aplikasi backend yang dibangun menggunakan Express, TypeScript, dan Prisma. Aplikasi ini menyediakan API untuk mengelola pengguna, termasuk registrasi, login, top-up saldo, dan transaksi.
